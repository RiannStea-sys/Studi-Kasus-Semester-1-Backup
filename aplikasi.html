Aplikasi Coding

Informasi tentang aplikasi coding Visual Studio Code dan Dev C++

Visual Studio Code

Visual Studio Code (VS Code) adalah editor kode sumber yang dikembangkan oleh Microsoft untuk Windows, Linux, dan macOS. Fitur termasuk dukungan untuk debugging, penyorotan sintaks, penyelesaian kode cerdas, snippet, refactoring kode, dan Git terintegrasi.

Fitur Utama:

  • IntelliSense untuk penyelesaian kode cerdas
  • Debugging terintegrasi
  • Dukungan Git bawaan
  • Ekstensibilitas melalui marketplace

Sublime

Sublime adalah aplikasi yang sering digunakan oleh mahasiswa informatika

Fitur Utama:

  • mudah digunakan
  • bisa menjadi pilihan kedua setelah visual studio code
  • Editor dengan penyorotan sintaks
  • Manajemen proyek

Dev C++

Dev-C++ adalah lingkungan pengembangan terpadu (IDE) gratis untuk pemrograman dalam C dan C++. Ini menggunakan port MinGW dari GCC (GNU Compiler Collection) sebagai kompilernya dan dapat digunakan untuk menulis, mengkompilasi, menautkan, dan menjalankan program C dan C++.

Fitur Utama:

  • Kompiler GCC berbasis MinGW
  • Debugger terintegrasi
  • Editor dengan penyorotan sintaks
  • Manajemen proyek
